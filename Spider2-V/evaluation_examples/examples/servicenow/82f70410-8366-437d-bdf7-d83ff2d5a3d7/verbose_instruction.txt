In this task, we are required to create a filter in the current Users page. To do this:
1. Find and locate the funnel icon at the top part (the middle one of 3 icons on the left of text "Users"). Click it.
2. Note that, there are already two filter conditions, we need to firstly empty these options by clicking the cross symbols on the rightmost of each row.
3. Then, find the input box "-- choose field --", click it.
4. In the drop-down menu, type the keyword "Employee" into the input box to quickly filter the results. Click the entry "Employee number" in the filtered list.
5. Next, we choose the operator for this option, click and choose "is" value
6. And in the input box on the right, just leave it empty
7. Click the button "AND" on the right to add another condition.
8. Again, we repeat the previous steps 3-6, to add another filter condition:
Choose option: Company
Choose operator: is
Input value: ACME Japan
9. Click the button "AND" again to add another condition:
Choose option: Default perspective
Choose operator: is
Input value:
(leave this field empty)
10. Click the button "AND" again to add another condition:
Choose option: State / Province
Choose operator: is
Input value:
(leave this field empty)
11. Click the button "AND" again to add the last condition:
Choose option: Enable Multifactor Authentication
Choose operator: is
Select value: false
12. Finally, click the button "Run" to filter the results. You may need to wait for a few seconds for the filter engine to work.
Till now, we have completed this task.
